# MBA 케이스 스터디 챗봇 시나리오 (Q&A형)

## 컨셉
케이스를 제시하고, 3개의 핵심 질문(Q1~Q3)을 던지고, 사용자가 직접 답변한 뒤, 그 답변을 실제 토론에서 어떻게 풀어낼 수 있을지 고민하게 하는 구조.

---

## 전체 흐름

```
[1. 프로필 입력] → [2. 맞춤 케이스 제시] → [3. Q1~Q3 제시]
                                                   ↓
                  [5. 토론 가이드]    ←    [4. 답변 작성]
```

---

## Phase 1: 프로필 입력

```
🤖: 안녕하세요! 맞춤형 경영 케이스를 드리기 위해
    몇 가지만 여쭤볼게요.

    1. 어떤 업종에서 일하고 계세요?
    2. 직급/역할이 어떻게 되세요?
    3. 경력은 몇 년 정도 되셨어요?

👤: IT 회사, 마케팅 팀장, 12년차요.

🤖: 감사합니다! 맞춤 케이스를 준비할게요.
```

---

## Phase 2: 맞춤 케이스 제시

사용자 프로필에 맞게 변환된 케이스를 제공한다.

```
🤖: 케이스를 준비했습니다. 천천히 읽어보세요.

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
📋 CASE: 넥스트소프트의 마케팅 전환점

넥스트소프트는 직원 450명 규모의 B2B SaaS 기업이다.
최근 3년간 매출 성장률이 15%에서 5%로 둔화되었고,
리드 전환율도 하락 추세다.

마케팅 팀장 김도현(12년차)은 AI 기반 마케팅 자동화
도구 도입을 검토하고 있다. 경쟁사 B사는 6개월 전
유사 도구를 도입해 전환율을 30% 끌어올렸다.

그러나 팀원 8명 중 5명이 도입에 반대하고 있다.
반대 측 주장은 "문제는 도구가 아니라 예산 부족"이다.
CEO는 다음 달 경영회의에서 마케팅 혁신안 보고를
요구하고 있으며, 연간 예산은 2억원으로 제한되어 있다.

[참고 데이터]
• 리드 전환율 추이: 22년 8.5% → 23년 6.2% → 24년 4.1%
• B사 도입 후 전환율: 4.8% → 6.2% (6개월)
• AI 도구 도입 비용: 연 1.2~3억원 (도구별 상이)
• 팀원 반대 비율: 찬성 3명 / 반대 5명
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

🤖: 다 읽으시면 말씀해주세요.
    3가지 질문을 드리겠습니다.
```

---

## Phase 3: Q1~Q3 제시

케이스에 대한 3가지 핵심 질문을 한 번에 제시한다.
질문은 난이도와 사고 유형이 다르게 설계한다.

```
🤖: 이 케이스에 대해 3가지 질문을 드립니다.
    하나씩 답변해주세요.

    Q1. 김도현 팀장이 직면한 핵심 문제는 무엇인가요?
        (단순히 "AI 도입 여부"가 아닐 수 있습니다)

    Q2. 당신이 김도현 팀장이라면 어떤 결정을 내리겠습니까?
        그 결정의 근거 3가지를 함께 제시해주세요.

    Q3. 만약 당신의 결정이 실패한다면,
        가장 큰 원인은 무엇이 될까요?

    Q1부터 시작해볼까요?
```

### 질문 설계 원칙

| 질문 | 사고 유형 | 의도 |
|------|----------|------|
| Q1 | 문제 정의 | 표면적 문제 너머 본질을 파악하는 힘 |
| Q2 | 의사결정 + 논거 | 결정을 내리고 근거를 구성하는 힘 |
| Q3 | 리스크 인식 | 자기 판단의 약점을 스스로 점검하는 힘 |

---

## Phase 4: 답변 작성

사용자가 Q1~Q3에 순서대로 답변한다.
각 답변 후 AI는 깊이를 유도하는 짧은 후속 질문을 1개만 던진다.

```
🤖: Q1. 김도현 팀장이 직면한 핵심 문제는 무엇인가요?

👤: 팀원들의 반대를 어떻게 설득할지가 핵심 문제인 것 같아요.

🤖: "설득"이 핵심이라고 보셨군요.
    혹시 설득 말고 다른 접근도 가능할까요?

👤: 음... 반대 의견이 맞는지 먼저 검증해보는 것도
    방법일 수 있겠네요. 정말 예산 문제일 수도 있으니까.

🤖: 좋습니다. 그러면 Q2로 넘어가겠습니다.
    당신이 김도현 팀장이라면 어떤 결정을 내리겠습니까?
    근거 3가지와 함께 답해주세요.

👤: 소규모 파일럿을 먼저 돌리겠습니다.
    근거 1) 전면 도입은 예산 초과 리스크
    근거 2) 반대 팀원도 결과를 보면 납득할 수 있음
    근거 3) CEO에게 단계적 계획을 보고하면 시간 확보 가능

🤖: 구체적인 근거네요.
    파일럿 기간은 어느 정도로 생각하세요?

👤: 2개월이요. 찬성하는 3명으로 먼저 진행하고요.

🤖: 알겠습니다. 마지막 질문입니다.
    Q3. 이 결정이 실패한다면 가장 큰 원인은 무엇일까요?

👤: 파일럿 기간 내에 유의미한 성과가 안 나오는 경우요.
    2개월은 짧을 수 있으니까.

🤖: 성과 부재를 리스크로 보셨군요.
    혹시 성과 외에 다른 종류의 실패 가능성은 없을까요?

👤: 아... 반대 팀원 5명이 파일럿 자체를 방해하거나,
    소외감으로 팀 분위기가 나빠질 수도 있겠네요.

🤖: 좋습니다. 3개 질문에 대한 답변이 모두 끝났습니다.
```

## Phase 5: 토론 가이드

사용자의 답변을 정리하고, "이걸 실제 토론에서 어떻게 풀어낼 수 있을지" 가이드를 제공한다.

```
🤖: 당신의 답변을 정리해봤습니다.
    그리고 실제 토론 상황에서 이 답변을 어떻게
    활용할 수 있을지 함께 고민해보겠습니다.

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
📝 당신의 답변 요약

Q1 핵심 문제:
  처음에는 "팀원 설득"으로 봤으나,
  "반대 의견이 맞는지 검증"으로 확장함

Q2 결정:
  소규모 파일럿 (2개월, 찬성파 3명)
  → 근거: 예산 리스크, 결과로 납득, CEO 시간 확보

Q3 실패 요인:
  성과 부재 + 팀 분열 리스크
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

🤖: 이제 이 내용을 실제 토론 자리에서
    어떻게 펼칠 수 있을지 생각해봅시다.

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
💬 토론 가이드

■ 예상되는 반론과 대응

반론 1: "파일럿 2개월로 뭘 알 수 있나?"
→ 대응 포인트: B사도 초기 3개월은 성과 하락 후 반등.
  단, 측정 지표를 사전에 명확히 설정해야 설득력 있음.
  "전환율 자체보다 프로세스 효율성을 먼저 측정한다"고
  프레이밍하면 기간 부담을 줄일 수 있음.

반론 2: "찬성파만 하면 편 가르기 아닌가?"
→ 대응 포인트: "자원자 기반"으로 프레이밍하되,
  반대 팀원에게도 파일럿 관찰자/피드백 역할을 부여하면
  소외감을 줄일 수 있음. 이 부분을 사전에 설계해두면
  토론에서 "팀 분열" 반론에 미리 답할 수 있음.

반론 3: "예산 2억에 도구 비용 1.2억이면 나머지 업무는?"
→ 대응 포인트: 이 부분은 당신의 답변에서 다루지 않았음.
  토론 전에 기존 마케팅 예산 집행 내역을 정리하고,
  어떤 항목을 줄여서 도구 비용을 확보할지 준비 필요.

■ 토론에서 더 강해지려면

1. "문제 재정의"로 시작하기
   토론에서 바로 해결책을 말하지 말고,
   "이 문제의 본질이 뭔지 먼저 짚어보겠습니다"로
   시작하면 논의를 주도할 수 있음.

2. 숫자로 말하기
   "파일럿 해보겠습니다"보다
   "2개월, 3명, 리드 100건 기준, 전환율 5%→7% 목표"
   처럼 구체적 숫자를 제시하면 신뢰도가 올라감.

3. 반대 의견을 먼저 인정하기
   "예산 문제라는 의견도 일리가 있습니다.
   그래서 최소 비용으로 검증할 방법을 생각했습니다."
   → 반대파를 적으로 만들지 않는 화법.
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

🤖: 이 중에서 실제로 가장 쓸 수 있을 것 같은
    포인트가 있나요? 혹은 더 고민해보고 싶은 부분이
    있으면 말씀해주세요.
```

## 질문(Q1~Q3) 유형 라이브러리

케이스 카테고리별로 Q1~Q3를 다르게 설계한다.

### 전략 케이스
```
Q1. 이 기업이 직면한 가장 근본적인 문제는 무엇인가요?
Q2. 당신이라면 어떤 전략을 선택하겠습니까? 근거 3가지.
Q3. 경쟁사가 당신의 전략에 대응한다면 어떻게 되나요?
```

### 리더십 케이스
```
Q1. 이 상황에서 리더로서 가장 먼저 파악해야 할 것은?
Q2. 팀원/조직에 어떤 메시지를 전달하겠습니까?
Q3. 당신의 접근이 오히려 상황을 악화시킬 수 있는 시나리오는?
```

### 마케팅 케이스
```
Q1. 이 기업의 현재 포지셔닝에서 가장 취약한 점은?
Q2. 어떤 마케팅 전략을 제안하겠습니까? 타겟/메시지/채널 포함.
Q3. 이 전략의 ROI를 어떻게 측정하겠습니까?
```

### 윤리/딜레마 케이스
```
Q1. 이 상황에서 충돌하는 가치는 각각 무엇인가요?
Q2. 어떤 선택을 하겠습니까? 어떤 가치를 우선하는 건가요?
Q3. 반대 선택을 한 사람을 어떻게 설득하겠습니까?
```

## 케이스 변환 규칙 (기존과 동일)

사용자의 업종/직급/경력에 따라 원본 케이스를 변환한다.

```
입력: 사용자 프로필 + 원본 케이스 템플릿
변환:
  - 기업 → 사용자 업종의 기업
  - 주인공 → 사용자 직급과 유사한 인물
  - 규모 → 사용자 회사 규모에 맞게 조정
  - 데이터 → 해당 업종에 맞는 수치
출력: 맞춤형 케이스 + Q1~Q3
```

## 이전 시나리오들과 비교

| 항목 | 02_기존 시나리오 | 04_사고유도형 | 05_Q&A형 (이번) |
|------|-----------------|-------------|----------------|
| 구조 | 6단계, 복잡 | 6단계, 유동적 | 5단계, 단순 |
| 질문 수 | 라운드당 1~2개 (총 10개+) | 없음 (사용자가 만듦) | **3개 고정** |
| AI 역할 | 소크라테스 교수 | 거울 | **출제자 + 토론 코치** |
| 답변 후 | 다음 라운드로 이동 | 상황이 전개됨 | **후속 질문 1개 + 토론 가이드** |
| 결과물 | 점수 리포트 (78/100) | 사고 패턴 분석 | **답변 요약 + 토론 꿀팁** |
| 소요 시간 | 20~35분 | 30분+ | **10~15분** |
| 난이도 | 중~상 | 상 | **중** |
| 핵심 경험 | "AI 교수와 토론" | "내 사고를 발견" | **"생각 정리 + 토론 준비"** |
